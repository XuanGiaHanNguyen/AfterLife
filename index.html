Instructions

• Click grid cells to toggle between live and dead cell

• Use the clear button to reset population setup

• Click the "Simulate" button to play

How does it work?

Each colored cell represent live cell

Any live cell with fewer than 2 live neighbors dies due to underpopulation

Any live cell with 2 or 3 live neighbors lives on to the next generation

Any live cell with more than 3 live neighbors dies due to overpopulation

Any dead cell with exactly 3 live neighbors becomes a live cell, as if by reproduction
